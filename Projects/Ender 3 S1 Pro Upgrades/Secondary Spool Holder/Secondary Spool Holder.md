The [[Creality Ender 3 S1 Pro]] came with only 1 spool holder. For printing with more than one color it is more convenient to have a second spool holder. The machine supports adding a second one on the other side so I printed one.
![[No secondary spool holder.jpg]]
# Model
The model I used is from https://www.printables.com/model/158734-ender-3-s1-secondary-spool-holder
[[Ender 3 S1 Secondary Spool Holder.stl]]
